🚀 Sistema de Cadastro com Flask

Sistema web de cadastro desenvolvido com Python e Flask, integrando interface em HTML/CSS com banco de dados MySQL via XAMPP.

⸻

📌 Funcionalidades
	•	🔐 Login de usuários
	•	👤 Cadastro de usuários
	•	🧾 Cadastro de clientes
	•	🏢 Cadastro de fornecedores
	•	🔄 Operações CRUD (Create, Read, Update, Delete)

⸻

🛠️ Tecnologias utilizadas
	•	Python
	•	Flask
	•	HTML
	•	CSS
	•	MySQL (XAMPP)
